【最初に自分の作業ブランチを作成】
$ git pull --rebase origin main
  ※リモートに置いてあるmainブランチのデータを取り込む （自分の作業内容をローカルでコミットしてから）

$ git branch <<ブランチ名（苗字）>>
$ git switch <<ブランチ名>>
  ※ブランチを作成し、そこへ移動

【以下開発中に繰り返すセーブ作業（ローカルでコミット）】
$ git add src
$ git commit -m "<<作業内容のコメント>>"
  ※ワーキングエリアに保存してからローカルリポジトリにコミット

【開発がひと段落したらメインに合流させる】
$ git push origin <<ブランチ名>>
  ※ローカルリポジトリをリモートリポジトリにプッシュ（<<ブランチ名>> → mainブランチ　へのプルリクエスト）
